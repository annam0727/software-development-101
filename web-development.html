Front-end vs. back-end

Front-end is what users see and interact with (HTML, CSS, JavaScript). Back-end is the “behind the scenes” logic (servers, APIs, databases).

Basic front-end tools

  • HTML — structure
  • CSS — style
  • JavaScript — interaction
